Fun Stuff

Attached is an iframe for eNGE, an experimental PS1 emulator written in pure JavaScript. I have tested this with Rage Racer, and it seems to work just fine if you're using the software render version, which is what I linked. It doesn't seem to support controllers, though, so unless you get it working by messing with site permissions, have fun playing PS1 games with a keyboard.


How to use

1. Please please PLEASE make sure you're using Chrome or a Chromium-based browser. Trust me, I have tried it with Firefox and it does Not work.


2. Drag and drop your BIOS file into the black screen, and the big text that says "eNGE" should stop blinking red.


3. Drag and drop the BIN file for your game into the black screen, and it should play the PS1 startup, then load your game. For multi-BIN games, just drag and drop the biggest BIN file since that's the one with the game data and the rest are just audio tracks. The audio tracks should still work in this case, though.
