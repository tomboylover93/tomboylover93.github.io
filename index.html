    Fun Stuff

    Attached is an iframe for eNGE, an experimental PS1 emulator written in pure JavaScript. I have tested this with Rage Racer and WipEout 3 (Japanese & Special Edition), and it seems to work just fine if you're using the software render version, which is what I linked. It does support controllers, but apparently only on Firefox, however the audio doesn't work there.


    How to use

    1. Drag and drop your BIOS file into the black screen, and the big text that says "eNGE" should stop blinking red.


    2. Drag and drop the BIN file for your game into the black screen, and it should play the PS1 startup, then load your game. For multi-BIN games, just drag and drop the biggest BIN file since that's the one with the game data and the rest are just audio tracks. The audio tracks should still work in this case, though.
